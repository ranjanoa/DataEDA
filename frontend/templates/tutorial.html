01. Data Ingestion

Initialize the system by selecting your primary CSV or Excel datasets. The ingestion engine automatically identifies temporal markers to synchronize multiple streams into a unified process timeline.

02. Metadata Alignment

Provide an optional LEGEND.csv to map physical units and operating boundaries. This enables high-fidelity visualization with automated scaling and stability band projection.

03. Stability Analysis

The processing core detects "Stable States" (Q1-Q3) and identifies process anomalies (Shutdowns) automatically. Use the real-time statistics table to audit process reliability and skewness.

04. Visual Synthesis

Generate multi-dimensional charts including 3D Scatters and Correlation Heatmaps. Use the Custom Plot Creator to synthesize complex relationships between variables.
